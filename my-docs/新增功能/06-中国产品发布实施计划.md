| Web 管理后台   | ✅ 已完成   | React + Vite，管理功能完整  |
| iOS 应用       | ✅ 已完成   | Swift + SwiftUI             |
| Android 应用   | ✅ 已完成   | Kotlin + Compose            |
| Gateway 服务   | ✅ 已完成   | Node.js 后端服务            |
| 数据库系统     | ✅ 已完成   | PostgreSQL + Drizzle ORM    |

| 10   | 代码签名证书    | Windows必须 | 3-5 工作日   | 用户         |

### 3.2 ICP 备案详细要求

备案材料清单：

- 营业执照（三证合一，有效期内）
- 法人身份证正反面扫描件
- 网站负责人身份证正反面扫描件
- 域名证书（域名需实名认证，注册人与备案主体一致）
- WHOIS 信息截图
- 备案承诺书（云服务商提供模板）

备案流程：

1. 在云服务商（阿里云/腾讯云/华为云）提交备案申请
2. 云服务商初审（1-2 工作日）
3. 管局审核（3-20 工作日）
4. 备案通过，获取 ICP 备案号
5. 在网站/APP 显著位置展示备案号

### 3.3 软件著作权申请

申请材料：

- 计算机软件著作权登记申请表
- 软件文档（用户手册或详细设计说明书）
- 软件源代码（前2000行 + 后2000行，共4000行以上）
- 申请人身份证明材料（营业执照）

申请渠道：

- 中国版权保护中心：https://www.ccopyright.com.cn/
- 各省版权局

### 3.4 AI 算法备案要求

适用条件：

- 使用 AI 大模型提供生成式服务
- 面向中国境内用户

备案材料：

- 算法安全自评估报告
- 拟公示内容
- 算法安全责任人身份证明
- 备案承诺材料
- 技术文档（模型架构、训练数据来源等）
- 管理文档（内容审核机制、应急预案等）

备案系统：https://beian.cac.gov.cn/

### 3.5 AI 内容标识要求（2025年9月起强制）

根据《人工智能生成合成内容标识办法》(GB 45438-2025)：

显式标识：

- 文字、声音、图形等可感知方式提示用户内容为 AI 生成

隐式标识：

- 嵌入文件元数据，用于平台核验与监管追溯

技术实现：

- 需要在代码中实现 AI 生成内容的标识功能
- Claude 可协助完成此功能开发

## 6. 应用商店上架指南

### 6.1 Windows - 微软商店

优势：

- 个人开发者免费上架
- 非游戏应用 0 抽成
- MSIX 格式免费 CDN 托管
- 全球一键发布

上架流程：

1. 注册微软合作伙伴中心账号
2. 创建应用提交
3. 上传 MSIX/EXE 安装包
4. 填写应用信息和截图
5. 提交审核

注意事项：

- 需要 EV 代码签名证书消除 SmartScreen 警告
- 建议同时提供官网直接下载渠道

### 6.2 Android - 国内应用商店

主要渠道：
| 商店 | 开发者账号费用 | 软著要求 |
|------|----------------|----------|
| 华为应用市场 | 免费 | 必须 |
| 小米应用商店 | 免费 | 必须 |
| OPPO 软件商店 | 免费 | 必须 |
| vivo 应用商店 | 免费 | 必须 |
| 应用宝 | 免费 | 必须 |
| 360 手机助手 | 免费 | 必须 |

通用上架材料：

- 营业执照
- 软件著作权证书
- ICP 备案号
- 隐私政策 URL
- 应用截图（至少 3 张）
- 应用图标（512x512）
- 应用描述

### 6.3 iOS - App Store 中国区

上架要求：

- 苹果开发者账号（688元/年）
- ICP 备案号
- 隐私政策 URL
- 应用截图（各尺寸）
- 应用描述

注意事项：

- iOS 暂不强制要求软著
- 需要在 App 内显著位置展示 ICP 备案号
- AI 功能需要说明数据处理方式

## 7. 代码签名证书指南

### 7.1 证书类型选择

| 类型        | SmartScreen 信任 | 价格           | 推荐场景   |
| ----------- | ---------------- | -------------- | ---------- |
| OV 代码签名 | 需累积下载量     | 2000-3000元/年 | 个人开发者 |
| EV 代码签名 | 即时信任         | 5000-8000元/年 | 企业发布   |

推荐：企业发布建议选择 EV 代码签名证书

### 7.2 推荐 CA 机构

国际：

- DigiCert
- GlobalSign
- Sectigo (Comodo)

国内：

- 沃通 (WoTrus)

### 7.3 申请材料

- 营业执照
- 法人身份证
- 企业银行账户信息（用于验证）
- 办公地址证明（租赁合同/水电账单）

## 8. 国内 AI 模型接入方案

### 8.1 推荐模型

| 模型     | 提供商   | 特点       |
| -------- | -------- | ---------- |
| 通义千问 | 阿里云   | 综合能力强 |
| 文心一言 | 百度     | 中文理解好 |
| MiniMax  | MiniMax  | 对话能力强 |
| 智谱 GLM | 智谱AI   | 开源友好   |
| 讯飞星火 | 科大讯飞 | 语音集成好 |

### 8.2 接入策略

方案一：替换默认模型

- 将默认 AI 模型从 Claude 替换为国内模型
- 优点：完全合规
- 缺点：能力可能有差异

方案二：多模型支持（推荐）

- 保留 Claude 作为可选项
- 默认使用国内模型
- 用户可在设置中切换
- 优点：灵活性高
