================================================================
  OpenClaw Windows 打包测试环境
================================================================

本目录包含三种 Windows EXE 打包方案的完整测试项目。

��� 目录结构
----------------------------------------------------------------
windows-packaging/
├── pkg-test/          # PKG 打包测试 (推荐)
├── nexe-test/         # NEXE 打包测试 (轻量)
├── electron-test/     # Electron 打包测试 (GUI)
├── TEST-GUIDE.txt     # 详细测试指南
├── quick-test.bat     # 快速测试脚本 (Windows)
└── README.txt         # 本文件

��� 快速开始
----------------------------------------------------------------

方式 1: 使用快速测试脚本 (推荐)
   双击运行: quick-test.bat
   按提示选择要测试的打包方式

方式 2: 手动测试单个方案
   
   PKG 测试:
   1. cd pkg-test
   2. npm install fs-extra
   3. node build.js
   4. dist\openclaw.exe
   
   NEXE 测试:
   1. cd nexe-test
   2. npm install fs-extra
   3. node build.js
   4. dist\openclaw.exe
   
   Electron 测试:
   1. cd electron-test
   2. node build.js (会自动安装依赖)
   3. npm start (开发模式)
      或 运行 dist\win-unpacked 中的 exe

�� 查看详细指南
----------------------------------------------------------------
打开 TEST-GUIDE.txt 查看:
- 三种方案的详细对比
- 测试步骤说明
- 常见问题解答
- 最佳实践建议

⚙️ 系统要求
----------------------------------------------------------------
- Node.js >= 18.0.0
- Windows 10/11
- 2GB+ 可用磁盘空间
- 稳定的网络连接（首次打包需下载依赖）

��� 推荐建议
----------------------------------------------------------------
- CLI 应用: 使用 PKG (平衡体积和稳定性)
- GUI 应用: 使用 Electron (完整桌面体验)
- 轻量部署: 使用 NEXE (文件最小)
