# OpenClaw Assistant 更新日志

记录 OpenClaw Assistant Windows 客户端的版本更新历史。

---

## [0.1.0] - 2026-02-07

### 首个公开版本

这是 OpenClaw Assistant Windows 客户端的首个公开发布版本。

### 新增功能

#### 核心功能
- **AI 对话**: 与 AI 助理进行自然语言交互
- **敏感操作确认**: 执行敏感操作前需用户确认，保护数据安全
- **设备配对**: 安全的设备认证和配对机制

#### 文件管理
- 浏览本地文件系统
- 文件/文件夹的创建、删除、移动、复制
- 文件搜索功能
- 快速访问常用目录

#### 系统监控
- 实时 CPU 和内存使用率监控
- 磁盘空间使用情况
- 进程列表查看和管理
- 系统信息展示

#### 技能系统
- 技能管理界面
- 技能商店浏览和安装
- 技能启用/禁用控制
- 内置文件整理技能

#### 审计日志
- 操作历史记录
- 日志筛选和搜索
- 日志导出功能

#### 订阅管理
- 订阅计划展示
- 使用量统计
- 配额管理

#### 设置选项
- Gateway 连接配置
- 外观主题设置（深色/浅色/跟随系统）
- 通知设置
- 隐私设置
- 快捷键配置
- 软件更新设置

#### 系统集成
- 系统托盘支持
- 开机自启动（可选）
- 自动更新机制
- Windows NSIS 安装包

### 技术特性

- 基于 Electron 28 构建
- React 18 + TypeScript 前端
- WebSocket 实时通信
- electron-updater 自动更新

## 版本号说明

版本号格式: `主版本.次版本.修订版本`

- **主版本**: 重大功能更新或不兼容的 API 变更
- **次版本**: 新功能添加，向后兼容
- **修订版本**: Bug 修复和小改进
