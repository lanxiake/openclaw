## 安装问题

### Q: 安装时提示"Windows 已保护你的电脑"怎么办？

**A:** 这是 Windows SmartScreen 的安全提示。点击"更多信息"，然后点击"仍要运行"即可继续安装。这是因为应用尚未获得 Windows 代码签名证书。

### Q: 安装需要管理员权限吗？

**A:** 默认安装到用户目录不需要管理员权限。如果选择安装到 `Program Files` 目录，则需要管理员权限。

### Q: 如何完全卸载应用？

**A:**
1. 通过"控制面板" > "程序和功能"卸载
2. 或运行安装目录下的 `Uninstall OpenClaw Assistant.exe`
3. 如需清除配置数据，删除 `%APPDATA%\OpenClaw Assistant` 目录

### Q: 便携版和安装版有什么区别？

**A:**
- **安装版**: 创建开始菜单快捷方式、桌面图标，支持自动更新
- **便携版**: 无需安装，可放在 U 盘使用，配置保存在程序目录

## 连接问题

### Q: 无法连接到 Gateway 怎么办？

**A:** 请按以下步骤排查：

1. **检查 Gateway 是否运行**
   - 确认 Gateway 服务已启动
   - 检查 Gateway 监听的端口是否正确

2. **检查网络连接**
   - 确认网络正常
   - 检查防火墙是否阻止了连接

3. **检查地址格式**
   - 确保使用正确的 WebSocket 地址格式：`ws://host:port`
   - 如使用 HTTPS，地址应为：`wss://host:port`

4. **检查认证信息**
   - 如果 Gateway 需要认证，确保已完成设备配对

### Q: 连接经常断开怎么办？

**A:**
1. 检查网络稳定性
2. 在设置中调整重连间隔和最大重连次数
3. 如果使用远程 Gateway，检查服务器状态

### Q: 设备配对失败怎么办？

**A:**
1. 确保 Gateway 地址正确
2. 检查 Gateway 是否允许新设备配对
3. 在 Gateway 管理端确认配对请求
4. 如多次失败，尝试重置设备后重新配对

### Q: 如何更换 Gateway 服务器？

**A:**
1. 断开当前连接
2. 在设置中修改 Gateway 地址
3. 如需要，取消当前设备配对
4. 连接新的 Gateway 并完成配对

## 功能问题

### Q: AI 助理没有响应怎么办？

**A:**
1. 检查是否已连接到 Gateway
2. 查看状态栏的连接状态
3. 尝试重新连接
4. 检查 Gateway 服务是否正常运行

### Q: 如何让 AI 执行文件操作？

**A:** 直接用自然语言描述您的需求，例如：
- "帮我把桌面上的图片移动到图片文件夹"
- "删除下载文件夹中超过30天的文件"
- "搜索文档中包含'报告'的 Word 文件"

### Q: 敏感操作确认对话框太频繁怎么办？

**A:** 敏感操作确认是为了保护您的数据安全。您可以：
1. 在设置中调整确认级别（不推荐完全关闭）
2. 对于信任的操作，可以选择"记住此选择"

### Q: 技能安装失败怎么办？

**A:**
1. 检查网络连接
2. 确认订阅计划是否支持该技能
3. 检查磁盘空间是否充足
4. 尝试重新安装

### Q: 如何导出聊天记录？

**A:**
1. 进入"审计日志"页面
2. 选择要导出的时间范围
3. 点击"导出"按钮
4. 选择导出格式（JSON/CSV）

## 性能问题

### Q: 应用启动很慢怎么办？

**A:**
1. 检查是否有杀毒软件在扫描
2. 尝试关闭"启动时自动连接"选项
3. 清理应用缓存数据
4. 确保系统有足够的可用内存

### Q: 应用占用内存过高怎么办？

**A:**
1. 关闭不需要的技能
2. 清理聊天历史记录
3. 重启应用
4. 检查是否有技能异常运行

### Q: 界面卡顿怎么办？

**A:**
1. 在设置中关闭动画效果
2. 减少同时打开的功能页面
3. 检查系统资源使用情况
4. 尝试重启应用

## 安全问题

### Q: 我的数据安全吗？

**A:** OpenClaw Assistant 采取多项措施保护您的数据：
- 所有通信使用加密传输
- 敏感操作需要用户确认
- 聊天记录仅保存在本地
- 不会上传您的个人文件

### Q: 如何保护我的隐私？

**A:**
1. 在设置中关闭"发送匿名使用统计"
2. 定期清理聊天历史
3. 设置较短的历史保留天数
4. 不要在对话中输入敏感信息（如密码）

### Q: 认证 Token 泄露了怎么办？

**A:**
1. 立即在设置中取消设备配对
2. 在 Gateway 管理端撤销该设备的访问权限
3. 重新进行设备配对获取新 Token

### Q: 如何查看 AI 执行了哪些操作？

**A:** 所有操作都记录在"审计日志"中，您可以：
1. 查看操作历史
2. 筛选特定类型的操作
3. 导出日志进行审计

## 更新问题

### Q: 如何检查更新？

**A:**
1. 进入"设置" > "软件更新"
2. 点击"检查更新"按钮
3. 如有新版本，会显示更新信息

### Q: 自动更新失败怎么办？

**A:**
1. 检查网络连接
2. 确认有足够的磁盘空间
3. 尝试手动下载安装包更新
4. 检查是否有杀毒软件阻止

### Q: 如何回退到旧版本？

**A:**
1. 卸载当前版本
2. 从官方渠道下载旧版本安装包
3. 安装旧版本
4. 注意：回退可能导致配置不兼容

### Q: 更新后设置丢失了怎么办？

**A:** 正常情况下更新不会丢失设置。如果发生：
1. 检查 `%APPDATA%\OpenClaw Assistant` 目录
2. 如有备份，恢复 `settings.json` 文件
3. 如无备份，需要重新配置
