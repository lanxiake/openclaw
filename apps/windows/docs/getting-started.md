## 系统要求

- **操作系统**: Windows 10 (64位) 或更高版本
- **内存**: 至少 4GB RAM (推荐 8GB)
- **存储空间**: 至少 500MB 可用空间
- **网络**: 需要网络连接以使用 AI 功能

## 安装步骤

### 方式一：使用安装程序 (推荐)

1. 下载最新版本的安装程序 `OpenClaw-Assistant-Setup-x.x.x-x64.exe`
2. 双击运行安装程序
3. 选择安装语言（支持简体中文和英文）
4. 阅读并同意许可协议
5. 选择安装目录（默认为 `C:\Program Files\OpenClaw Assistant`）
6. 点击"安装"开始安装
7. 安装完成后，可选择立即启动应用

### 方式二：使用便携版

1. 下载便携版 `OpenClaw-Assistant-x.x.x-portable.exe`
2. 将文件放置到您希望的目录
3. 双击运行即可使用，无需安装

## 首次启动

启动 OpenClaw Assistant 后，应用会：

1. **最小化到系统托盘**: 应用默认在后台运行，您可以在系统托盘找到 OpenClaw 图标
2. **显示主窗口**: 点击托盘图标或使用快捷键打开主窗口

### 系统托盘操作

- **单击**: 显示/隐藏主窗口
- **右键**: 打开菜单
  - 显示窗口
  - 连接/断开 Gateway
  - 退出应用

## 连接 Gateway

OpenClaw Assistant 需要连接到 Gateway 服务才能使用 AI 功能。

### 本地 Gateway

如果您在本地运行 Gateway：

1. 确保 Gateway 服务已启动
2. 在侧边栏的"Gateway 连接"区域
3. 默认地址为 `ws://localhost:18789`
4. 点击"连接"按钮

### 远程 Gateway

如果连接远程 Gateway：

1. 在地址栏输入 Gateway 的 WebSocket 地址
2. 如需认证，请先完成设备配对
3. 点击"连接"按钮

### 设备配对

首次连接需要进行设备配对：

1. 进入"设置" > "Gateway 连接"
2. 输入 Gateway 地址
3. 点击"发起配对"
4. 在 Gateway 管理端确认配对请求
5. 配对成功后会自动保存认证信息

## 基本操作

### 对话功能

1. 在主界面的对话区域输入您的问题或指令
2. 按 Enter 或点击发送按钮
3. AI 助理会处理您的请求并回复

**示例指令**:
- "帮我整理桌面上的文件"
- "查看系统内存使用情况"
- "搜索文档文件夹中的 PDF 文件"

### 文件管理

1. 点击侧边栏的"文件管理"
2. 浏览本地文件系统
3. 支持的操作：
   - 查看文件列表
   - 创建/删除文件夹
   - 移动/复制文件
   - 搜索文件

### 系统监控

1. 点击侧边栏的"系统监控"
2. 查看系统信息：
   - CPU 使用率
   - 内存使用情况
   - 磁盘空间
   - 运行中的进程

### 技能管理

1. 点击侧边栏的"技能管理"
2. 查看已安装的技能
3. 从技能商店安装新技能
4. 启用/禁用技能
