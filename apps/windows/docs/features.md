## 对话功能

对话是与 AI 助理交互的主要方式。

### 发送消息

1. 在输入框中输入您的问题或指令
2. 按 `Enter` 发送，或按 `Shift + Enter` 换行
3. 等待 AI 助理处理并回复

### 对话历史

- 对话记录会自动保存在本地
- 可以在侧边栏查看历史对话
- 支持搜索历史消息

### 敏感操作确认

当 AI 助理需要执行敏感操作时（如删除文件、执行命令），会弹出确认对话框：

- **操作名称**: 显示将要执行的操作
- **风险等级**: 低/中/高
- **操作描述**: 详细说明操作内容
- **确认/拒绝**: 您可以选择允许或拒绝该操作

### 常用指令示例

| 类别 | 示例指令 |
|------|----------|
| 文件操作 | "帮我整理下载文件夹" |
| 系统查询 | "查看当前内存使用情况" |
| 应用启动 | "打开记事本" |
| 文件搜索 | "搜索文档中的 PDF 文件" |
| 进程管理 | "显示占用内存最多的进程" |

## 文件管理

文件管理功能让您可以浏览和操作本地文件系统。

### 浏览文件

1. 点击侧边栏的"文件管理"
2. 默认显示用户主目录
3. 双击文件夹进入，点击面包屑导航返回

### 文件操作

| 操作 | 说明 |
|------|------|
| 新建文件夹 | 点击工具栏的"新建文件夹"按钮 |
| 删除 | 选中文件/文件夹，点击删除按钮 |
| 重命名 | 右键点击文件，选择"重命名" |
| 复制/移动 | 选中文件，使用复制/剪切，然后粘贴 |
| 搜索 | 在搜索框输入关键词，支持通配符 |

### 快速访问

侧边栏提供常用目录的快速访问：
- 桌面
- 文档
- 下载
- 图片

## 系统监控

实时监控系统资源使用情况。

### 系统概览

- **CPU 使用率**: 实时显示 CPU 占用百分比
- **内存使用**: 已用/总量，使用百分比
- **磁盘空间**: 各分区的使用情况

### 进程管理

1. 查看所有运行中的进程
2. 按 CPU/内存 使用率排序
3. 结束指定进程（需要确认）

### 系统信息

显示详细的系统信息：
- 操作系统版本
- 计算机名称
- 处理器信息
- 已安装内存

## 技能系统

技能是扩展 AI 助理能力的模块。

### 已安装技能

1. 点击侧边栏的"技能管理"
2. 查看已安装的技能列表
3. 每个技能显示：
   - 名称和图标
   - 版本号
   - 启用状态
   - 简要描述

### 技能操作

| 操作 | 说明 |
|------|------|
| 启用/禁用 | 切换技能的启用状态 |
| 配置 | 修改技能的配置参数 |
| 卸载 | 移除已安装的技能 |
| 更新 | 更新到最新版本 |

### 技能商店

1. 点击"技能商店"标签
2. 浏览可用技能：
   - 按分类筛选
   - 搜索技能名称
   - 查看技能详情
3. 点击"安装"添加技能

### 内置技能

| 技能名称 | 功能描述 |
|----------|----------|
| 文件整理 | 自动整理和分类文件 |
| 系统清理 | 清理临时文件和缓存 |
| 应用启动器 | 快速启动常用应用 |
| 剪贴板管理 | 管理剪贴板历史 |

## 审计日志

记录所有 AI 助理执行的操作。

### 查看日志

1. 点击侧边栏的"审计日志"
2. 查看操作记录列表
3. 每条记录包含：
   - 时间戳
   - 操作类型
   - 操作详情
   - 执行结果

### 筛选日志

- **按时间**: 选择日期范围
- **按类型**: 筛选特定操作类型
- **按状态**: 成功/失败/待确认

### 导出日志

1. 点击"导出"按钮
2. 选择导出格式（JSON/CSV）
3. 选择保存位置

## 订阅管理

管理您的订阅计划和使用配额。

### 订阅计划

| 计划 | 功能 |
|------|------|
| 免费版 | 基础功能，每日限制对话次数 |
| 专业版 | 更多对话次数，高级技能 |
| 团队版 | 多设备支持，优先支持 |
| 企业版 | 无限制，API 访问，专属支持 |

### 使用量统计

- 今日对话次数
- 本月 AI 调用次数
- 存储空间使用

### 升级订阅

1. 点击侧边栏的"订阅管理"
2. 查看当前计划和可用计划
3. 选择目标计划
4. 完成支付流程

## 设置选项

### Gateway 连接

- **Gateway 地址**: WebSocket 服务器地址
- **认证 Token**: 用于身份验证
- **自动连接**: 启动时自动连接
- **重连设置**: 重连间隔和最大次数

### 外观主题

- **主题模式**: 深色/浅色/跟随系统
- **主色调**: 选择界面主色
- **字体大小**: 小/中/大
- **动画效果**: 启用/禁用界面动画

### 通知设置

- **启用通知**: 开启/关闭通知
- **通知声音**: 播放提示音
- **消息预览**: 在通知中显示内容
- **桌面通知**: 系统级通知

### 隐私安全

- **保存聊天历史**: 是否保存对话记录
- **历史保留天数**: 自动清理旧记录
- **使用统计**: 发送匿名使用数据

### 快捷键

| 功能 | 快捷键 |
|------|--------|
| 发送消息 | Enter |
| 新建对话 | Ctrl + N |
| 切换侧边栏 | Ctrl + B |
| 打开设置 | Ctrl + , |

### 软件更新

- **自动检查更新**: 定期检查新版本
- **自动下载**: 发现更新后自动下载
- **预发布版本**: 接收测试版更新
